# **App Name**: EduTrack

## Core Features:

- Student Management: Add, update, delete, and view student details including first name, last name, email, and department.
- Course Management: Add and view available courses, including course name, course code, and credits.
- Enrollment Management: Assign courses to students, creating enrollment records to track which students are in which courses.
- Course Roster Retrieval: Retrieve all students enrolled in a specific course.
- Student Enrollment Retrieval: Retrieve all courses a student is enrolled in.
- Mark Management: Update or view student marks for a specific course.
- Firebase Authentication: Allow students to log in to view their records securely using Firebase Authentication.

## Style Guidelines:

- Primary color: Soft blue (#77B5FE), reflecting trust and education.
- Background color: Light gray (#F0F4F8), providing a clean, uncluttered backdrop.
- Accent color: Warm orange (#FFB347), used sparingly to highlight important actions.
- Body font: 'Inter', sans-serif, for a clean and modern feel.
- Headline font: 'Space Grotesk', sans-serif, for titles and important headers, and 'Inter' for body text.
- Use consistent, professional icons representing academic elements. 
- Clean and intuitive layout with clear separation of sections for easy navigation.
